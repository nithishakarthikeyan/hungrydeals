# hungrydeals
Local Restaurant Discovery Website 

Hungry Deals is a full-stack web application that displays restaurants in Winnipeg using real-time data from the Yelp Fusion API. The project was built to gain hands-on experience with REST APIs, backend development, and dynamic frontend rendering.

# Features

1. Fetches real restaurant data from the Yelp Fusion API

2. Displays restaurant name, rating, price range, address, categories, and images

3. Responsive layout with custom typography

4. RESTful backend built with FastAPI

5. Frontend built using vanilla JavaScript, HTML, and CSS

# Tech Stack

Backend: Python 3.9, FastAPI, Requests, Yelp Fusion API

Frontend: HTML, CSS, JavaScript (Fetch API)

# What I Learned

- Working with third-party APIs and authentication

- Designing RESTful backend services with FastAPI

- Handling asynchronous data fetching on the frontend

- Debugging real-world issues such as port conflicts and API latency

- Building clean, reusable UI components without frameworks
